---

## Install

```bash
$ npm install fy -g
```

## Usage

Translation data is fetched from [iciba.com](http://iciba.com) and [fanyi.youdao.com](http://fanyi.youdao.com),
and only support Chinese and English translation.

Translate one word.

```bash
$ fy love
```
